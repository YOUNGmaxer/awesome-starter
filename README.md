# awesome-starter

收录前端各种技术栈的最新最佳项目启动模板。

一些模板能力要求：

* 提供 VSCode 断点调试能力（附带配置）
* 快速简单的开发启动
* 生产构建
* 代码规范最佳实践

## Chrome 插件

- [ ] 极简JS版：JS
- [ ] 极简TS版：TS
- [ ] Vue 版：Vue3 + Vite

## Web 纯前端

- [ ] Vue3 + Vite5 + Node20 + pnpm

## Web 全栈

- [ ] Vue3 + Vite5 + Nestjs + Node20 + pnpm

## 桌面端

- [ ] Electron + Vue3 + Vite5
